# Mini Projeto de Análise de Dados

Dados de 2019 de uma empresa de prestação de serviços.

- CadastroFuncionarios
- CadastroClientes
- BaseServiçosPrestados

### Objetivo:

1. Valor Total da Folha Salarial<br>
2. Faturamento da empresa<br>
3. % de funcionários que já fechou algum contrato<br>
4. Total de contratos que cada área da empresa já fechou<br>
5. Total de funcionários por área<br>
6. Ticket médio mensal dos contratos<br>
