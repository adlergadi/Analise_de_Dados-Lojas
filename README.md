# Miniprojeto de Análise de Dados

## Descrição

Realizar análises de dados de uma empresa fictícia utilizando Python e bibliotecas de análise de dados.

A partir de diferentes bases de dados contendo informações sobre funcionários, clientes e serviços prestados, o script tem como objetivo:

- Valor total da folha salarial
- Faturamento total da empresa
- Percentual de funcionários que fecharam contratos
- Quantidade de contratos fechados por área
- Distribuição de funcionários por área
- Ticket médio mensal dos contratos

## Estrutura dos Dados

### 1 - Cadastro de Funcionários

Arquivo: `CadastroFuncionarios.csv`

Contendo:

- ID do funcionário
- Área
- Salário base
- Benefícios
- Impostos
- Vale transporte (VT)
- Vale refeição (VR)

### 2️ - Cadastro de Clientes

Arquivo: `CadastroClientes.csv`

Contendo:

- ID do cliente
- Valor do contrato mensal

### 3️ - Base de Serviços Prestados

Arquivo: `BaseServiçosPrestados.xlsx`

Contendo:

- ID do funcionário responsável
- ID do cliente
- Tempo total de contrato (em meses)
